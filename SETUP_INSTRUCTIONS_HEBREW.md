## 🚀 הוראות התקנה והפעלה

### שלב 1: הפעלת Gmail API

1. היכנס ל-[Google Cloud Console](https://console.cloud.google.com/)
2. צור פרויקט חדש או בחר פרויקט קיים
3. חפש "Gmail API" בחיפוש
4. לחץ על "Enable" להפעלת ה-API
5. עבור ל-"Credentials" בתפריט הצדדי
6. לחץ על "Create Credentials" > "OAuth client ID"
7. בחר "Desktop app" כסוג האפליקציה
8. הורד את קובץ ה-credentials
9. שנה את שם הקובץ ל-`credentials.json`
10. העבר את הקובץ לתיקייה של הפרויקט

### שלב 2: התקנת הספריות הנדרשות

```bash
pip install -r requirements.txt
```

### שלב 3: הרצת התוכנית

```bash
python gmail_organizer.py
```

### שלב 4: אישור הרשאות

1. **יפתח חלון דפדפן** - התחבר לחשבון Gmail שלך
2. **אשר את ההרשאות** שהתוכנית מבקשת
3. התוכנית תתחיל לארגן את תיבת הדואר שלך

## ❓ בעיות נפוצות

### הבעיה: "credentials.json not found"
**פתרון:** ודא שהורדת את קובץ ה-credentials מ-Google Cloud Console

### הבעיה: "Permission denied"
**פתרון:** ודא שאישרת את כל ההרשאות בחלון הדפדפן

### הבעיה: "API not enabled"
**פתרון:** חזור לשלב 1 והפעל את Gmail API

## 💡 טיפים לשימוש

1. **הרץ את התוכנית פעם בשבוע** לארגון שוטף

3. **התאם אישית** את התיקיות לצרכים שלך
4. **גבה את הכללים** דרך Gmail Settings
