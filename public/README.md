# ⚠️ קובץ זה לא בשימוש!

הקובץ `public/index.html` הוא קובץ ישן שמציג דף "coming soon" והוא **לא משמש** את האפליקציה.

האפליקציה משתמשת ב-`web/index.html` במקום.

אם אתה רואה את דף ה-"coming soon" במקום את האפליקציה, זה אומר שהשרת משרת את הקובץ הזה במקום את האפליקציה React Native.

**פתרון:**
1. ודא שאתה מריץ את האפליקציה עם `expo start --web` או `npm run web`
2. אם אתה משתמש ב-Docker, ודא שאתה משתמש ב-`Dockerfile` ולא ב-`Dockerfile.static`
3. אם אתה רוצה למחוק את הקובץ הזה, אתה יכול למחוק את התיקייה `public/` כולה
